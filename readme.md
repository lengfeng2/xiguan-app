我想要开发一款习惯追踪APP 基于vue3 + element-plus框架构建，构建完需要编译并打开web浏览器，目的是为了用于实现自律打卡功能，同时管理个人目标和生活记录。该APP需要包含以下具体功能模块：
1. 用户登录与注册功能：用户需要先注册账号，才能使用APP的所有功能。
2. 习惯创建模块：允许用户创建、编辑和删除各类习惯项目。
3. 每日打卡模块：设置每日/每周/每月等不同周期的打卡目标，记录完成情况。
4. 数据统计和可视化图表：根据用户的打卡记录，生成统计图表，展示习惯完成情况、周期完成情况等。
5. 本地通知提醒：根据用户的打卡目标，设置本地通知提醒，帮助用户保持自律。
6. 暗色模式支持模块：提供暗色主题模式用于主题切换。
